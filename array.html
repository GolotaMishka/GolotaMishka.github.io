Array

  • array.every(callbackFn) – повертає true, якщо callbackFn повертає true для кожного елементу масиву

    [5, 6, 7, 8].every(item => item > 4) === true
    [5, 6, 7, 8].every(item => item > 6) === false
  • array.some(callbackFn) – повертає true, якщо callbackFn повертає true хоча б для одного елементу масиву

    [5, 6, 7, 8].some(item => item === 7) === true
    [-5, -3, -1, 2].some(item => item > 0) === true
    [-5, -3, -1, 2].some(item => item > 3) === false
  • array.filter(callbackFn) – створює новий масив з усіма елементами, які проходять тест, реалізований переданою callbackFn
    (якщо функція повертає true на ітерації, то елемент на цій ітерації буде записаний в новий масив, якщо false - ні)

    [4, -2, 4, -4, 1].filter(item => item > 0) -> [4, 4, 1]
    ["Mishka","Maks", "Svyat", "Dima"].filter(item => item.startsWith("M")) -> ["Mishka","Maks"]
  • array.forEach(callbackFn) – виконує задану функцію один раз для кожного елемента масиву(нічого не вовертає)

  • array1.concat(array2) – об’єднує два або більше масивів.
    Цей метод не змінює існуючі масиви, а повертає новий масив.

  • array.join(separator) – створює та повертає новий рядок, об’єднуючи всі елементи в масиві,
    розділяючи їх комами або вказаним рядком роздільника. Якщо масив містить лише один елемент,
    то цей елемент буде повернуто без використання роздільника.)
    ['K', 'L', 'M'].join() -> K,L,M
    ['Hello', 'beautiful', 'web'].join(', ') -> Hello, beautiful, web
  • array.sort() – сортує елементи масиву на місці і повертає відсортований масив. Порядок сортування
    за замовчуванням — висхідний (можна передати функцію, яка визначає порядок сортування),
    заснований на перетворенні елементів у рядки, а потім порівнянні їх
    послідовностей із значеннями одиниць коду UTF-16.
    ['K', 'L', 'M'].join() -> K,L,M
    ['Hello', 'beautiful', 'web'].join(', ') -> Hello, beautiful, web
  • array.reverse() – повертає масив на місце. Перший елемент масиву стає останнім, а останній елемент масиву стає першим.
    ['K', 'L', 'M'].reverse() -> ["M", "L", "K"]
  • array.slice(start, end) – повертає копію частини масиву в новий об’єкт масиву, вибраний від початку до кінця (кінець не включено),
    де початок і кінець представляють індекс елементів у цьому масиві. Початковий масив не буде змінено.
    ['K', 'L', 'M'].slice(2) -> ["M"]
    ['K', 'L', 'M'].slice(0, 1) -> ["K"]
    ['HTML', 'CSS', 'JS', 'React'].slice(1, 3) -> ["CSS", "JS"]
  • array.splice(start, deleteCount, item1, item2, itemN) – змінює вміст масиву, видаляючи або замінюючи елементи та/або додаючи нові елементи на їх місце. Щоб отримати доступ до частини масиву, не змінюючи його, див. slice().
    const months = ['Jan', 'March', 'April', 'June'];
    months.splice(1, 0, 'Feb');
    // inserts at index 1
    console.log(months);
    // expected output: Array ["Jan", "Feb", "March", "April", "June"]

    months.splice(4, 1, 'May');
    // replaces 1 element at index 4
    console.log(months);
    // expected output: Array ["Jan", "Feb", "March", "April", "May"]
  • array.includes(value) – визначає, чи містить масив певне значення серед своїх записів, повертаючи true чи false.
  • array.indexOf(element) – повертає перший індекс, за яким даний елемент можна знайти в масиві, або -1, якщо він відсутній.
    ['ant', 'bison', 'camel', 'duck', 'bison'].indexOf('bison') -> 1
    ['ant', 'bison', 'camel', 'duck', 'bison'].indexOf('wolf') -> -1
  • array.lastIndexOf(element, fromIndex) – повертає останній індекс, за яким даний елемент можна знайти в масиві, або -1, якщо він відсутній. Пошук у масиві здійснюється назад, починаючи з fromIndex.
    ['Dodo', 'Tiger', 'Penguin', 'Dodo'].lastIndexOf('Dodo') -> 3
    ['Dodo', 'Tiger', 'Penguin', 'Dodo'].lastIndexOf('Tiger') -> 1
  • array.map(function(element, index, arr) { /* ... */ }) – створює новий масив, заповнений результатами виклику функції для кожного елемента в масиві
    [1, 4, 9, 16].map(x => x * 2) -> [2, 8, 18, 32]
    ['HTML', 'CSS', 'JS', 'React'].map((tech) => tech.length) -> [4, 3, 2, 5]
  • array.pop() – видаляє останній елемент з масиву і повертає цей елемент. Цей метод змінює довжину масиву.
  • array.push(1,2) – додає один або кілька елементів в кінець масиву і повертає нову довжину масиву.
    const animals = ['pigs', 'goats', 'sheep'];

    const count = animals.push('cows');
    console.log(count);
    // expected output: 4
    console.log(animals);
    // expected output: Array ["pigs", "goats", "sheep", "cows"]

    animals.push('chickens', 'cats', 'dogs');
    console.log(animals);
    // expected output: Array ["pigs", "goats", "sheep", "cows", "chickens", "cats", "dogs"]
  • array.shift() – видаляє перший елемент із масиву та повертає цей видалений елемент. Цей метод змінює довжину масиву.
  • array.unshift(1,2) – додає один або кілька елементів на початок масиву і повертає нову довжину масиву
  • array.find(callbackFn) – повертає значення першого елемента в наданому масиві, яке задовольняє надану функцію тестування.
    Якщо жодне значення не задовольняє функції тестування, повертається undefined.
    [5, 12, 8, 130, 44].find(element => element > 10) -> 12
    [5, 12, 8, 130, 44].find(element => element === 17) -> undefined
  • array.reduce((previousValue, currentValue, currentIndex, array) => { /* ... */ }, initialValue) – виконує функцію зворотного виклику “reducer” для кожного елемента масиву,
    передаючи значення, що повертається з обчислення попереднього елемента.
    Кінцевим результатом запуску редюсера для всіх елементів масиву є одне значення.
    const array1 = [1, 2, 3, 4];
    const reducer = (previousValue, currentValue) => previousValue + currentValue;

    // 1 + 2 + 3 + 4
    console.log(array1.reduce(reducer));
    // expected output: 10

    // 5 + 1 + 2 + 3 + 4
    console.log(array1.reduce(reducer, 5));
    // expected output: 15
